# Public Directory

Place all static wedding assets (like `Groom.jpg`, `Bride.jpg`, and any `Album1.jpg` to `Album6.jpg`) directly in this directory so they are served at the root level by Vite.

For example:
- `/public/Groom.jpg` -> accessed in code as `"Groom.jpg"` or `"/Groom.jpg"`
- `/public/Bride.jpg` -> accessed in code as `"Bride.jpg"` or `"/Bride.jpg"`
